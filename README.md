# Case-Drop Core

## This project is mainly dedicated for my Minecraft Server, however it also can be used for other Paper-based servers.

### Features:
- AdminChat
- Alert
- Ban
- BanIP
- ChatManager
- Feed
- Freeze
- Fly
- Gamemode
- Godmode
- GetIP
- Kick
- MSG
- Mute
- Nick
- Ping
- Unban
- UUID getter
- Vanish
